# 📊 Excel Sales Analytics Project

## 📌 Project Overview

This project focuses on analyzing sales and financial performance using Advanced Microsoft Excel. The objective is to transform raw business data into meaningful insights that help stakeholders make informed business decisions.

The project was developed as part of the Codebasics Excel Analytics course and demonstrates practical business reporting using Excel.

---

# 🎯 Business Objectives

### Sales Analytics

- Analyze customer performance across different markets.
- Compare actual market performance against sales targets.
- Identify growth opportunities and underperforming regions.
- Support data-driven sales decisions.

### Finance Analytics

- Analyze Profit & Loss (P&L) statements by Fiscal Year.
- Analyze P&L performance by Markets.
- Analyze monthly P&L trends.
- Evaluate business profitability over different time periods.
